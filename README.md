# 📋 Gestor de Tareas — Flask CRUD

Aplicación web CRUD de gestión de tareas desarrollada con Python y Flask.

## Funcionalidades
- Crear, editar, eliminar y listar tareas
- Filtrar por estado (pendiente / completada)
- Prioridades (alta, media, baja)
- Dashboard con estadísticas

## Instalación
```bash
pip install -r requirements.txt
python app.py
```
Abrir en: http://127.0.0.1:5000

## Tecnologías
- Python 3 + Flask
- HTML5 / CSS3 / JavaScript
- JSON como almacenamiento
